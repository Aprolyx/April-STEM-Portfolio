# April’s STEM Educator Portfolio

## Overview
This website is a personal portfolio for a STEM educator. It includes multiple pages that share background, experience, and a way to contact the educator.

## How to Use
- Home page introduces the educator
- About page shows experience and background
- Contact page allows users to send a message

## Technologies Used
- HTML
- CSS
